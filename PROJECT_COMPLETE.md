## 🎯 **How to Use**

### **Start the Professional Server**
```bash
python vpn_server_professional.py
```

### **Start a Client**
```bash
python vpn_client_main.py
```

### **Access Dashboard**
Open browser: http://localhost:8081

### **Send Messages**
1. Go to server console
2. Type any message with spaces: "Hello world this works perfectly!"
3. Press Enter
4. Watch dashboard update in real-time
